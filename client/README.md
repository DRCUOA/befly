# Client

Vue 3 + Tailwind CSS frontend for the Rambulations writing platform.

## Features

- Browse and read essays with a calm, literary aesthetic
- Dark mode with system preference detection and manual toggle
- Theme-based essay navigation (click a theme to see all related essays)
- Responsive layout with keyboard accessibility and reduced-motion support
- Cover image uploads with crop and reposition
- Typography suggestions (non-blocking, inline)
- Word count on pause
- Admin dashboard (user management, content moderation, analytics)

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Runs on port 5178.

## Build

```bash
npm run build
```

## Type Checking

```bash
npm run type-check
```

## Tests
